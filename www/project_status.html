Last Week's Activities

It looks like we have checkouts, commits and updates working now, when talking with a local repository (ra_local). (The `status' command works too, in a simple way.) Yes, there are a few little bugs still. But for the most part, the general internal machinery appears to be working. This is great news.

Next week's agenda:

  • throw ra_dav into the action

  • start expanding the interfaces so that commits and updates can take multiple targets.

M2 is a week from Sunday, and we're cruising right towards it.
