Last Week's Activities

This week, we continued to flesh out repository filesystem functionality. We also started revamping client-side code in preparation to directly link with the filesystem. If all goes well, then in the upcoming week should see "first contact" between the two halves; instead of XML, the client will be sending commits into a real repository!

After that, the plan is to get updates working, which is a a bit more complicated than commits. Once the client and repository are interacting correctly, we'll making our final "push" toward Milestone 2: inserting Greg Stein's DAV layer into the system.
