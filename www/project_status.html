Recent Activities

We've been self-hosting for about five weeks now, and things are going very well. We've been working on M4/M5 in parallel (see milestones above), and activity has been very high on the developer's mail list.

After hitting M4/M5, we'll begin work on Alpha. The big Alpha task is adding copy/move/merge (a.k.a. branch and tag) functionality to the client libraries.

If you did something and we forgot to list it here, please send a patch to kfogel@collab.net, or change this file yourself.
