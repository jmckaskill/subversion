Project Subversion Source Code

The source code is under (very) active development right now. You can:

  • Download the source code with CVS
  • Browse the CVS repository over the web


Standard "tar.gz" and other download formats will be made available as we approach release; for now, the CVS tree is the best way to get Subversion.

Contributing Source Code

To contribute code to Subversion:

  • Join the developer community mailing lists: (see the Members page.)
  • The Subversion project more-or-less abides by the guidelines used by the Apache APR project. Please review these.
  • After checking out a copy of the source, read the file HACKING at the top of the tree. It describes coding and logging conventions, as well as how to build.
