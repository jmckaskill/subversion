Missing runtime file

The file msvcp60.dll are not found in your system folder and you must download and install it before using Subversion.
msvcp60.dll are one of the VC runtime files.

Instructions for download and installing:

1. Please, select (click on) one of the links below (depending on your language) and download the file to your disk:
  • VC6RedistSetup_enu.exe (Default/English)
  • VC6RedistSetup_deu.exe (German)
  • VC6RedistSetup_jpn.exe (Japanese)
2. Run your downloaded VC6RedistSetup_xxx.exe:
  1. You must accept the License
  2. Choose "Browse" for extracting your files. A file with the name vcredist.exe are now placed in the folder you choosed.
  3. Run vcredist.exe. You must restart your computer when finished.

If you need more info about the VC Runtime files, please read from this link:

  • http://support.microsoft.com/support/kb/articles/Q259/4/03.ASP
