Missing system file

The file shfolder.dll are not found in your system folder and you must download and install it before using Subversion.

Instructions for download and installing:

  1. Download shfinst.EXE wich includes shfolder.dll to your disk from this link.
  2. Run the downloaded shfinst.EXE:
    1. After accepting the license, you can extract to a location on your hard disk.
    2. Hit the Unzip button after that you have decided where to unzip the files.
  3. Open the the folder where you choosed to extract the file to in Windows Explorer and open the x86 folder.
  4. Just run ShFolder.Exe and your done.

You are now ready to use Subversion (if you are not nold to restart your machine during the installation process).
